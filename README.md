Cotton University CS & IT Help Desk

This project is a student-developed web portal created for the students of the Department of Computer Science & Information Technology, Cotton University.

The website was built to make important academic and admission-related information easily accessible in one place. It includes admission resources, merit lists, placement-related links, FAQs, notices, and other useful information for students.

Features
Responsive design for mobile and desktop
Admission and merit list information
FAQ section
Quick links to important resources
Visitor counter integration
Clean and simple user interface
Smooth animations and modern layout
Technologies Used
HTML
CSS
JavaScript
Font Awesome
